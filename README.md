# ESP32 PCB Design Project

## Project Description
This project is a custom PCB design using ESP32 for electronics engineering requirements. The PCB includes power regulation, GPIO pin connections, and basic communication interfaces.

## Features
- Compact PCB layout
- ESP32 microcontroller support
- Power regulation circuit
- GPIO header connections
- Easy soldering design

## Components Used
- ESP32 Dev Module
- Resistors
- Capacitors
- LED Indicator
- Push Buttons
- Voltage Regulator
- Pin Headers

## Software Used
- EasyEDA / KiCad
- GitHub

## Repository Structure

```text
gerber/   -> Gerber manufacturing files
images/   -> PCB screenshots and 3D views
source/   -> Editable PCB source files
```
